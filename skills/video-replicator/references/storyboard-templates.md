# Storyboard Templates Reference

**Version:** v2.29
**Last Updated:** 2026-02-04

This reference documents the 8 narrative templates available for the Storyboard Grid feature. Each template provides a 9-panel structure optimized for cinematic storytelling.

## Quick Reference

| Template | ID | Emotional Arc | Best For |
|----------|----|--------------:|----------|
| Dialogue/Confrontation | `dialogue-confrontation` | Tension → Breaking point → Resolution | Arguments, negotiations, ultimatums |
| Chase/Pursuit | `chase-pursuit` | Urgency → Escalation → Climax | Escapes, races, time pressure |
| Discovery/Reveal | `discovery-reveal` | Mystery → Investigation → Revelation | Plot twists, secrets uncovered |
| Journey/Transformation | `journey-transformation` | Beginning → Struggle → Arrival | Travel, personal growth |
| Romance/Connection | `romance-connection` | Distant → Drawing closer → Together | Meeting, bonding, intimacy |
| Comedy/Surprise | `comedy-surprise` | Setup → Complication → Punchline | Mishaps, visual gags |
| Horror/Suspense | `horror-suspense` | Calm → Unease → Terror | Dread, threat, survival |
| Product Story | `product-story` | Problem → Discovery → Transformation | Product demos, testimonials |

## Template 1: Dialogue/Confrontation

**ID:** `dialogue-confrontation`
**Arc:** Tension → Breaking point → Resolution
**Best For:** Arguments, negotiations, tense decisions, ultimatums
**Min Characters:** 2

### 9-Panel Structure

| Panel | Shot Type | Purpose | Motion | Character |
|-------|-----------|---------|--------|-----------|
| 1 | Wide Establishing | Context, isolation, scale | Slow pan or static | Both |
| 2 | Medium (Character A) | Introduce character state | Subtle push in | A |
| 3 | Medium (Character B) | Counter-position, conflict | Subtle push in | B |
| 4 | Tight Close-up A | Emotional beat, reaction | Static or micro-push | A |
| 5 | Insert Shot | Narrative detail, symbol | Static | None |
| 6 | Tight Close-up B | Counter-reaction, peak | Static or micro-push | B |
| 7 | High-Angle Wide | Vulnerability, stakes | Slow crane or static | Both |
| 8 | Two-Shot | Relationship dynamics | Track or static | Both |
| 9 | Wide Exit | Resolution, commitment | Pull out or static | Both |

### Dialogue Hints

- Panel 1: [silence - establishing shot]
- Panel 2: Opening statement, sets the conflict
- Panel 3: Response, disagreement or challenge
- Panel 4: [emotional reaction] Internal conflict visible
- Panel 5: [silence] or [narrator] describes significance
- Panel 6: [desperate/angry] Key emotional moment
- Panel 7: [narrator] or [silence] - perspective shift
- Panel 8: Direct confrontation, key exchange
- Panel 9: [resolute] Final decision or action

## Usage

### CLI Commands

```bash
# List all templates
python scripts/storyboard_templates.py --list

# Show detailed template
python scripts/storyboard_templates.py --show dialogue-confrontation

# Suggest templates for a premise
python scripts/storyboard_templates.py --suggest "Two people argue about survival"

# JSON output
python scripts/storyboard_templates.py --show product-story --json
```

### In generate_storyboard.py

```bash
# Interactive wizard
python scripts/generate_storyboard.py --project my-story

# Non-interactive
python scripts/generate_storyboard.py \
  --project my-story \
  --template dialogue-confrontation \
  --premise "Two survivors face a critical decision" \
  --character-ids 27,28 \
  --aspect-ratio 16:9 \
  --yes
```

### Python API

```python
from storyboard_templates import (
    get_template,
    list_templates,
    suggest_template,
    get_panel_prompt_hints,
)

# Get a specific template
template = get_template("dialogue-confrontation")
print(template.name)  # "Dialogue/Confrontation"
print(template.emotional_arc)  # "Tension → Breaking point → Resolution"

# Get panel details
panel_4 = template.get_panel(4)
print(panel_4.shot_type)  # "Tight Close-up (Character A)"
print(panel_4.default_motion)  # "Static or micro-push"

# Suggest templates based on premise
suggestions = suggest_template("Two people argue about leaving")
# Returns: ["dialogue-confrontation", "journey-transformation"]

# Get prompt hints for image/motion generation
hints = get_panel_prompt_hints(template, 4)
# Returns: {"image_hints": {...}, "motion_hints": {...}}
```

## Customizing Templates

Templates can be modified in two ways:

### 1. Panel Overrides (Runtime)

Pass `panel_overrides` to `StoryboardConfig`:

```python
config = StoryboardConfig(
    project="my-story",
    premise="...",
    template_id="dialogue-confrontation",
    panel_overrides={
        5: {
            "shot_type": "Product Close-up",
            "purpose": "Show the item they're arguing about",
        }
    }
)
```

### 2. Custom Templates (Code)

Add new templates to `storyboard_templates.py`:

```python
MY_CUSTOM = StoryTemplate(
    template_id=TemplateType.CUSTOM,  # Add to enum
    name="My Custom Template",
    description="Description...",
    use_case="Use cases...",
    emotional_arc="Start → Middle → End",
    panels=[
        PanelDefinition(1, "Shot Type", "Purpose", "Motion", "A", "Dialogue hint"),
        # ... 8 more panels
    ]
)
```

## Best Practices

1. **Match template to story** - Use `suggest_template()` as a starting point
2. **Two characters minimum** for Dialogue and Romance templates
3. **Insert shots (Panel 5)** are character-free - use for objects, symbols, environment
4. **Dialogue hints** guide facial expressions in grid generation
5. **Default motions** are suggestions - Veo may interpret based on content
6. **Aspect ratio** applies to all panels - choose before generation
