# Prompt Templates by Scene Type

*Last Updated: 2026-01-13*

## Character/Model Scenes

### Fashion/Luxury

```
[Subject description], wearing [outfit], in [environment],
[pose/action], high-end fashion photography lighting,
[camera angle], 8K, luxury editorial, Vogue aesthetic,
no text, no logos, clean plate
```

**Example:**
```
A confident 30-year-old Asian woman with sleek black hair,
wearing an elegant black silk dress, in a minimalist beige
photography studio with seamless paper backdrop, turning
slowly toward camera with direct eye contact, high-key
studio lighting with large softbox, medium close-up shot
with shallow depth of field, 8K, luxury editorial,
Vogue aesthetic, no text, no logos, clean plate
```

### Lifestyle/Casual

```
[Subject description], in [natural environment],
[authentic action], natural lighting [time of day],
[camera style], candid, authentic, lifestyle photography,
no text, no logos, clean plate
```

### Fitness/Sports

```
[Subject description], in [gym/outdoor setting],
[athletic action], dynamic lighting, [camera movement],
energetic, powerful, sports photography, motion blur,
no text, no logos, clean plate
```

## Product Scenes

### Hero Shot (Product Alone)

```
[Product name/description], centered on [surface/backdrop],
[arrangement], professional product photography lighting
with [specific setup], [camera angle], commercial quality,
8K, sharp focus, no text, no logos, clean plate
```

**Example:**
```
Premium glass skincare bottle with gold cap, centered on
white marble surface with soft shadows, single product hero
shot, professional three-point product lighting with rim
light, eye-level straight-on shot, commercial quality, 8K,
sharp focus, no text, no logos, clean plate
```

### Product in Use

```
[Subject] using/holding [product], in [environment],
[action with product], [lighting], [camera], lifestyle
product photography, authentic, no text, no logos,
clean plate
```

### Product Detail/Macro

```
Extreme close-up of [product detail], showing [texture/feature],
macro photography, focus stacking, [lighting],
product detail shot, 8K, sharp, no text, no logos
```

## Environment/Scene Setup

### Studio Settings

```
Professional [color] photography studio, [backdrop type],
[lighting equipment visible/hidden], [props if any],
clean minimal aesthetic, commercial grade
```

**Backdrops:**
- Seamless paper: beige, white, black, gray
- Cyclorama wall (infinity curve)
- Textured fabric backdrop
- Gradient backdrop

### Natural Environments

```
[Location type], [time of day], [weather/atmosphere],
[specific details], natural lighting, [mood],
cinematic, establishing shot potential
```

### Interior Settings

```
[Room type], [design style], [key furniture/props],
[window/artificial lighting], [color palette],
interior photography, architectural detail
```


```
Camera slowly dollies [in/out], [subject action],
smooth continuous motion, cinematic
```

### Pan

```
Camera pans [left/right] revealing [elements],
[subject action], steady movement
```

### Tracking/Follow

```
Camera tracks [subject] as they [action],
following movement, dynamic
```

### Orbit

```
Camera slowly orbits around [subject],
revealing [different angles], smooth rotation
```

## Lighting Keywords by Mood

### Luxury/High-End
- High-key studio lighting
- Large softbox, minimal shadows
- Rim lighting for separation
- Even, flattering skin tones

### Dramatic/Moody
- Low-key lighting
- Strong shadows, chiaroscuro
- Single hard light source
- High contrast

### Natural/Authentic
- Golden hour
- Window light, soft and directional
- Practical lights visible
- Naturalistic, unforced

### Technical/Product
- Three-point lighting
- Light tent/diffused
- Gradient backgrounds
- Controlled reflections

## Metatoken Libraries

### Style Categories

**Luxury/Fashion:**
```
luxury, high-end, editorial, Vogue aesthetic, refined,
sophisticated, elegant, premium, exclusive
```

**Commercial/Advertising:**
```
commercial grade, advertising photography, campaign quality,
professional, polished, brand-appropriate
```

**Documentary/Authentic:**
```
documentary style, candid, raw, authentic, BTS,
unscripted feel, genuine, naturalistic
```

**Cinematic:**
```
cinematic, film grain, anamorphic, movie still,
widescreen, dramatic, theatrical
```

### Technical Quality

```
8K, ultra high resolution, sharp focus, professional grade,
photo-realistic, highly detailed, masterful composition
```

### Negative Prompts (What to Avoid)

```
no text, no logos, no watermarks, no overlays,
no graphics, no UI elements, no borders,
clean plate, unobstructed
```
