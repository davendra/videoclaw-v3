# Finding Winning Ads to Copy

*Last Updated: 2026-01-13*

## Core Philosophy

> "If a brand pays for 3 months, it's profitable."

Ads that run for 90+ days are proven winners. Organic reels with 10x typical views have cracked the format. Find these patterns, extract what works, rebuild with your assets.

## Method A: Ad Longevity (MagicBrief)

**Best for**: Finding proven paid ads

1. Go to [MagicBrief](https://magicbrief.com) or similar ad library tool
2. Filter ads by "Run Time"
3. Look for ads running **90+ days**
4. If a brand keeps paying, the ad is profitable

**What to look for**:
- Consistent creative format across multiple ads
- High production value that justifies long runtime
- Clear product/brand focus

## Method B: Performance Estimate (MagicBrief)

**Best for**: Quick discovery of high-performers

1. Use the "Winners" filter in MagicBrief
2. Algorithm analyzes 60+ attributes:
   - Hook effectiveness
   - Retention patterns
   - Engagement signals
3. Shows likely profitable ads

**Combine with Method A**: Winners filter + 60+ day runtime = highest confidence

## Method C: Organic Viral (Instagram/TikTok)

**Best for**: Finding viral organic formats

### Instagram Process

1. Go to trendsetting brand profile (Nike, Gymshark, etc.)
2. Install "Sort Feed" Chrome extension
3. Sort Reels by view count (highest first)
4. Look for **outliers**: 36M views when typical is 1M

### TikTok Process

1. Search brand or product hashtag
2. Filter by "Most Liked" or "Most Viewed"
3. Identify videos with 10x+ typical engagement
4. Note the format patterns

**Key indicators**:
- View count significantly above brand's average
- High comment-to-view ratio
- Format being copied by other creators

## Method D: Facebook Ad Library

**Best for**: Competitor research, free

1. Go to [Facebook Ad Library](https://www.facebook.com/ads/library)
2. Search competitor brand name
3. Filter by "Active" ads
4. Sort by longest running

**Limitations**: No performance data, but longevity = success

## Method E: TikTok Creative Center

**Best for**: TikTok-specific formats

1. Go to [TikTok Creative Center](https://ads.tiktok.com/business/creativecenter)
2. Browse "Top Ads" section
3. Filter by industry/region
4. Analyze top performers

## What Makes an Ad Worth Copying

### Green Flags

- Running 90+ days (paid ads)
- 10x+ typical engagement (organic)
- Clear, simple structure
- Strong hook in first 2 seconds
- Product clearly featured
- Format already being copied by others

### Red Flags

- Celebrity/influencer dependent (can't replicate)
- Heavily text-reliant (text doesn't translate to other languages)
- Brand-specific humor (won't work for your brand)
- Complex VFX (expensive to recreate)
- Trending audio (may be dated by production time)

## Quick Selection Checklist

Before copying an ad, verify:

- [ ] Structure is reproducible (no unique talent/location)
- [ ] Product integration is clear and adaptable
- [ ] Duration matches your needs (15s, 30s, 60s)
- [ ] Aesthetic fits your brand
- [ ] Hook can be adapted to your product
- [ ] No heavy reliance on copyrighted music/elements
