# Video Stage Directors

VideoClaw should use short, operational stage guides instead of a large orchestration framework.

## Brief Director

Goal:

1. turn user intent into a concrete production brief

Must decide:

1. production mode
2. target platform
3. backend intent
4. duration target
5. success definition

## Storyboard Director

Goal:

1. turn the brief into a usable shot or beat plan

Must decide:

1. scene order
2. beat pacing
3. continuity requirements
4. dialogue intent

## Asset Director

Goal:

1. produce or collect the assets each scene needs

Must record:

1. scene-to-asset mapping
2. backend used
3. failed assets and retry reasons

## Review Director

Goal:

1. evaluate whether the current output is good enough to proceed

Must check:

1. visual quality
2. character consistency
3. artifact completeness
4. subtitle or text contamination

## Publish Director

Goal:

1. package the final deliverable and make readiness explicit

Must record:

1. final output path
2. variant information
3. unresolved issues
4. publish readiness or blocker
