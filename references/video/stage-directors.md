# Video Stage Directors

## Brief Director

Must define:

1. production mode
2. target platform
3. backend intent
4. duration target
5. success definition

## Storyboard Director

Must define:

1. scene order
2. beat pacing
3. continuity requirements
4. dialogue intent

## Asset Director

Must record:

1. scene-to-asset mapping
2. backend used
3. failed assets
4. retry reasons

## Review Director

Must check:

1. visual quality
2. character consistency
3. artifact completeness
4. subtitle or text contamination

## Publish Director

Must record:

1. final output path
2. variants
3. unresolved issues
4. publish readiness
