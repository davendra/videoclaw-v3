# Character Reference Sheet

Identity sheets are the continuity anchor for director-mode character work.

Recommended coverage:

1. at least one `identity` role reference for every character-bearing scene
2. two or more identity references when the character appears across multiple scenes
3. `characterName` set on character-specific identity sheets
4. explicit bindings to the scenes where the character appears

Use outfit, material, environment, motion-camera, and palette-mood sheets for supporting continuity, but do not treat them as substitutes for identity coverage.

Readiness warnings should stay non-blocking unless a character-bearing director scene has no bound Identity Sheet at all.
